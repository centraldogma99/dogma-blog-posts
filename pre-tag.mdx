`<pre>` 태그는 preformatted tag를 의미한다.

`<pre>` 태그 안의 텍스트는 고정폭 폰트가 적용되며, 띄어쓰기와 줄바꿈이 그대로 유지된다(원래는, `<br>`같은 것을 써야 줄바꿈이 된다).
즉, 소스 코드에 작성한 텍스트가 그대로 출력된다.

`<code>` 태그와 함께 사용되는 경우가 많다.

```html
<pre>
  <code>
    const f = () => {
      console.log('hello, world!')
    }

    f();
  </code>
</pre>
```
